# CNN
*Also known as Convolutional Neural Network or Conv Net.*

It is very popular and is used for images specifically,  grid like structures. It works very well on problems like Image Classification, Object Detection, etc.

The problem with Linear classifiers or fully connected network is that they destroy the spatial structure of the input and convert it into 1-D vector, but for image the spatial structure is very important. Thats why we cannot use those for problems related to images.
